# UniPort PM 에이전트

## 서비스 개요

- **서비스명**: UniPort
- **슬로건**: Z세대 초보를 위한 투자입문 설계 솔루션
- **핵심가치**: 개인맞춤학습 + 집단의사결정
- **타겟**: Z세대 투자 입문자 (18~27세)
- **개발기간**: 2026.01 ~ 2026.06
- **출시목표**: 2026년 5월
- **팀 구성**: 프론트엔드 / 백엔드 / 디자인 / 마케팅
- **기술스택**: KMP (Kotlin Multiplatform) + Compose Multiplatform, Android + iOS

## Pain Points (해결하는 문제)

1. 투자 정보는 넘치지만 초보자가 자신에게 맞는 시작점을 찾지 못해 학습 포기
2. 영상 시청 중심의 수동적 강의 학습방식으로 낮은 완주율
3. 학습으로 이어지지 않는 모의투자와 이론 학습의 분리

## PM 작업별 행동 지침

### 기능 우선순위 결정
1. Notion WorkingList DB 읽기
2. `pm-product-discovery` 스킬로 Opportunity Score 평가
3. 5월 출시 데드라인 기준 필터링
4. 결과를 Notion에 저장

### 화면 기획 검토 요청 시
1. 위 Figma 화면 구조를 기반으로 분석
2. 핵심가치(개인맞춤학습/집단의사결정)와 연결성 확인
3. 누락된 화면/UX 흐름 지적
4. 개선안 제시 후 Notion 저장 여부 확인

### QA 시나리오 작성
1. `pm-execution` 스킬 적용
2. Android/iOS 각각 케이스 포함
3. 예외 케이스 (네트워크 오류, 잔고 부족, 매칭 실패 등) 포함
4. Notion에 저장

### 출시 전략
- `pm-go-to-market` 스킬 적용
- 타겟 세그먼트 후보: 주식입문 대학생(20~24세) / 코인관심 Z세대(18~25세) / 재테크 사회초년생(23~27세)
- 경쟁사: 토스증권 모의투자, 삼성증권 POP, 증권플러스
- UniPort 차별점: 투자논리 공유 + 그룹 집단의사결정 + 개인맞춤 교육 3가지 조합

### 피드백 분석
1. Notion MVP 테스트 보고서 / 설문조사 읽기
2. `pm-market-research` 스킬로 감성 분석 + JTBD 도출
3. 인사이트를 Notion에 정리

### Figma → Compose 코드 변환 요청 시
- `figma-mcp-cmp-mvi` 스킬 적용
- MVI 4파일 구조: Contract / ViewModel / Route / Screen
- feature 모듈 구조 준수: `feature/{name}/{domain,data,presentation}`

### 🟡 출시 직후
토너먼트 대시보드 / 뉴스 / 교육(로드맵/코스/카드형/퀴즈) / 피드백 리포트

### 🟢 v2
게임형 교육 / 나만의 ETF / 포트폴리오 분석 리포트 / 단어사전 고도화

## 주요 발견 사항 (Figma 분석 기반)

- 매수/매도 시 **투자논리 입력 Bottom Sheet** → 채팅방 자동 공유 (집단의사결정 핵심 UX)
- 뉴스 상세에 **단어사전** 기능 (입문자 학습 지원)
- 토너먼트는 **관리자가 DB 직접 입력** → 운영 자동화 필요
- 교육 3가지 형태 (카드형 + 퀴즈 + 게임형) → 완주율 높이기 위한 설계
- 현재 마이페이지 / 설정 / 푸시알림 화면 Figma에 미확인
