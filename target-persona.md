# UniPort 타겟 페르소나

## 주요 세그먼트 3가지

---

### Persona A — "주식 입문 대학생" (20~24세)

**프로필**
- 이름: 김지수 (22세, 경영학과 3학년)
- 월 용돈/아르바이트: 70~100만원
- 현재 자산: 100~300만원 (적금, 현금)

**상황**
- 유튜브/릴스로 주식 영상 자주 봄
- "해봐야 하는 거 알겠는데 어디서 시작해야 할지 모르겠어"
- 친구들이랑 주식 얘기는 하지만 실제 투자는 안 해봄

**Pain Point**
- 정보는 넘치는데 내 상황에 맞는 게 없음
- 잘못 투자해서 잃을까봐 무서움
- 혼자 공부하다가 질려서 포기

**UniPort 가치**
- 진단 → 내 성향 파악 → 맞춤 학습 로드맵
- 모의투자로 리스크 없이 연습
- 같은 입문자들끼리 그룹 투자 — 덜 외롭고 배울 수 있음

---

### Persona B — "코인 관심 Z세대" (18~25세)

**프로필**
- 이름: 이민준 (21세, 자퇴 후 프리랜서)
- 코인은 해봤지만 주식은 낯섦
- 빠른 수익 기대가 있음

**상황**
- 코인으로 소액 수익/손실 경험 있음
- 주식이 더 안정적이라고 생각하지만 진입 장벽 느낌
- 커뮤니티 기반 정보 탐색에 익숙 (디스코드, 레딧, 에펨코리아)

**Pain Point**
- 주식은 공부할 게 너무 많아 보임
- 나 혼자 분석하는 게 맞는지 불안함
- 유의미한 수익을 내는 방법을 모름

**UniPort 가치**
- 그룹 채팅 + 투자논리 공유 → 커뮤니티 익숙한 사람에게 딱 맞음
- 토너먼트 → 게임처럼 접근 가능한 경쟁 요소
- 뉴스 + 종목 연결 → 정보 탐색 습관 활용

---

### Persona C — "재테크 사회초년생" (23~27세)

**프로필**
- 이름: 박소현 (25세, 취업 2년차, 마케터)
- 월 소득: 250~300만원
- 현재: 적금 + ISA 계좌 정도만 가입

**상황**
- "재테크는 해야 하는데 어떻게 해야 할지 몰라"
- 주식은 관심 있지만 분석할 시간/여유 없음
- 친구/직장 동료 추천으로 앱 설치해보는 경우 많음

**Pain Point**
- 시간 없어서 직접 공부 못함
- 투자 잘못해서 원금 날릴까봐 걱정
- 전문가 말 믿어야 하나 아니면 내가 해야 하나 혼란

**UniPort 가치**
- 진단으로 내 성향/리스크 수준 파악 → 자신감
- 그룹 포트폴리오 → 혼자가 아니라 집단 지성으로 결정
- 교육 로드맵 → 바쁜 중에 짬짬이 학습

---

## 공통 인사이트

- 3 페르소나 모두 **"정보 과잉 + 시작점 부재"** 문제 공유
- **친구/커뮤니티의 영향**을 크게 받음 → 바이럴/그룹 기능이 핵심
- 앱 설치 트리거: 친구 추천 > SNS 광고 > 검색
- 이탈 위험: 처음 1~2주 안에 "이거 나한테 필요한가?" 느끼면 삭제
