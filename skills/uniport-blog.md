# UniPort 블로그 포스트 자동 생성

SEO 최적화된 블로그 포스트를 완성형으로 작성한다. 키워드 리서치부터 본문, 메타데이터까지 한 번에 출력한다.

## 시작 전 컨텍스트 확인

반드시 먼저 읽는다:
- `marketing/brand-guide.md` — 브랜드 보이스, 금지 표현
- `marketing/target-persona.md` — 타겟 페르소나
- `marketing/product-faq.md` — 제품 핵심 기능

## 입력 수집

제공되지 않은 경우에만 물어본다:

1. **메인 키워드**: 어떤 키워드로 SEO 최적화 (없으면 `/uniport-keyword-analysis` 먼저 제안)
2. **주제/각도**: 어떤 방향으로 쓸지 (예: "주식 초보 가이드", "모의투자 비교", "Z세대 재테크")
3. **플랫폼**: 네이버 블로그 / 브런치 / 티스토리 / 미디엄
4. **길이**: 짧음(1000자) / 기본(1500자) / 길게(2000자+)
5. **타겟 페르소나**: A(대학생) / B(코인 Z세대) / C(사회초년생)

## 블로그 포스트 제작 워크플로우

### Step 1: 키워드 전략 설정

**메인 키워드**: 검색량 + 경쟁도 균형
**서브 키워드**: 메인과 연관된 2~3개
**LSI 키워드**: 자연스럽게 포함할 관련 용어

삽입 위치:
- 제목에 메인 키워드 포함 (앞부분)
- 첫 단락에 메인 키워드 1회
- 소제목 2~3개에 서브 키워드
- 본문 전체에 밀도 2~3% 유지

### Step 2: 제목 공식

**검색량 최우선 (SEO)**:
- "[숫자] + 메인키워드 + (초보/입문/방법)"
  예: "주식 처음 시작하는 방법 5가지 (2026년 최신)"

**클릭율 최우선 (CTR)**:
- "[페르소나] + [공감] + [해결]"
  예: "20대 주식 시작하고 싶은데 어디서 시작해야 할지 모르겠다면"

**정보성 (Evergreen)**:
- "[개념] 뜻, 예시, 투자에 활용하는 법"
  예: "PER 뜻 쉽게 설명 (주식 초보 필독)"

3가지 제목 옵션 항상 제공

### Step 3: 본문 구조

**1000자 (짧은 포스트)**:
```
리드 문단 (150자) + 소제목 2~3개 × 본문 + 마무리 CTA
```

**1500자 (기본 포스트)**:
```
리드 문단 (200자)
소제목 1: 문제/배경
소제목 2: 핵심 정보 1
소제목 3: 핵심 정보 2
소제목 4: UniPort 자연스러운 연결
마무리 CTA (100자)
```

**2000자+ (심화 포스트)**:
```
리드 문단 + TL;DR 요약
소제목 1~5: 깊이 있는 정보
비교 표 또는 체크리스트 포함
마무리 + 관련 포스트 추천
```

### Step 4: UniPort 자연스러운 언급

**잘못된 방법** (광고처럼 느껴짐): "지금 바로 유니포트를 다운받으세요!"

**올바른 방법** (정보 흐름 속 자연스럽게):
- "이걸 실제로 연습해볼 수 있는 앱이 있어요. 유니포트라는 모의투자 앱인데..."
- "처음 시작하는 분들한테 유니포트 같은 그룹 모의투자 앱이 특히 도움돼요. 혼자 하는 것보다..."

### Step 5: CTA 공식

마무리 단락 기준:

- **부드러운 CTA**: "처음이라면 유니포트 모의투자로 먼저 경험해보는 걸 추천해요. (링크)"
- **강한 CTA**: "지금 바로 무료로 시작해보세요 → [앱스토어 링크]"
- **커뮤니티 CTA**: "댓글로 여러분의 첫 투자 경험을 공유해주세요"

### Step 6: 플랫폼별 최적화

**네이버 블로그**:
- 이미지 3~5장 삽입 권장 (alt 텍스트 키워드 포함)
- 내부 링크 2~3개 (같은 블로그 내 관련 포스트)
- 태그: 관련 키워드 10~15개

**브런치**:
- 스토리텔링 중심 문체
- 첫 문장이 특히 중요 (구독자에게 발송됨)
- 읽기 시간 표시 (1500자 ≈ 3분)

**티스토리**:
- SEO 플러그인 호환 형식
- 메타 설명 (160자)
- 구조화된 데이터 고려

```markdown
## 블로그 포스트: [주제]
**플랫폼**: [네이버/브런치/티스토리]
**메인 키워드**: [키워드]
**예상 읽기 시간**: [분]

---

### 📌 제목 옵션 3가지

### 🏷️ 메타 설명 (160자)
[완성된 메타 설명]

---

### 📄 본문

**[선택된 제목]**

[리드 문단]

## [소제목 1]

## [소제목 2]
[본문]

## [소제목 3]
```

## 품질 체크리스트

- [ ] 메인 키워드가 제목 앞부분에 포함
- [ ] 첫 단락에 키워드 자연스럽게 포함
- [ ] 소제목에 서브 키워드 분산
- [ ] 투자 수익 보장 표현 없음
- [ ] Z세대 언어 (전문용어 번역됨)
- [ ] UniPort 언급이 광고스럽지 않음
- [ ] CTA 명확
- [ ] 1500자 이상 (기본 기준)

## 콘텐츠 시리즈 기획

장기 SEO를 위한 시리즈 포스트 제안:

**주식 입문 시리즈** (5편):
1. "주식 처음 시작하는 법 완벽 가이드"
2. "주식 용어 10개 — 이것만 알면 됨"
3. "분산투자란? 초보가 꼭 알아야 하는 이유"
4. "모의투자로 시작해야 하는 이유"
5. "그룹 투자 vs 혼자 투자 — 어느 게 더 좋을까"
