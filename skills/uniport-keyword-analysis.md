# UniPort 키워드 트렌드 분석

UniPort 마케팅 콘텐츠에 활용할 키워드를 채널별로 분석하고 우선순위를 제시한다.

## 시작 전 컨텍스트 확인

반드시 먼저 읽는다:
- `marketing/brand-guide.md` — 경쟁 포지셔닝
- `marketing/target-persona.md` — 타겟 세그먼트

## 입력 수집

제공되지 않은 경우에만 물어본다:

1. **분석 목적**: 블로그 SEO / 유튜브 SEO / SNS 해시태그 / 광고 키워드
2. **주제 범위**: 투자 전반 / UniPort 기능 / 경쟁사 비교 / Z세대 재테크
3. **채널**: 네이버 / 구글 / 유튜브 / 인스타그램 / 틱톡 (복수 선택)

## 키워드 분석 워크플로우

### Step 1: 씨드 키워드 수집

UniPort 핵심 키워드 카테고리:

**제품 직접 키워드**
- 모의투자 앱, 주식 시뮬레이터, 가상 주식
- 그룹 투자, 함께 투자, 투자 커뮤니티
- 주식 입문, 주식 초보, 처음 주식

**타겟 키워드**
- 20대 재테크, 대학생 주식, 사회초년생 투자
- Z세대 투자, MZ 재테크
- 주식 시작 방법, 주식 배우기

**경쟁사 연관 키워드**
- 토스증권 모의투자, 삼성증권 POP
- 증권플러스 대안, 주식 앱 추천

**교육/학습 키워드**
- 주식 용어, PER 뜻, ETF 설명
- 분산투자, 포트폴리오 만들기
- 주식 공부 방법

### Step 2: 채널별 분석 기준

#### 네이버 검색 (블로그/포스트)

| 분류 | 분석 항목 | 도구 참고 |
|-----|---------|---------|
| 검색량 | 월간 검색 수 (PC+모바일) | 네이버 DataLab |
| 경쟁도 | 상위 노출 블로그 수 | 네이버 검색 직접 확인 |
| 성장성 | 최근 3개월 검색량 트렌드 | 네이버 DataLab 트렌드 |

**우선순위 공식**: 검색량 높음 + 경쟁도 낮음 + 성장세 = 최우선 타겟

#### 유튜브 SEO

| 분류 | 분석 항목 |
|-----|---------|
| 조회수 | 동일 키워드 영상 평균 조회수 |
| 구독자 경쟁 | 상위 채널 구독자 수 |
| 최신성 | 최근 3개월 내 콘텐츠 포화도 |
| 댓글 인사이트 | 시청자 자주 쓰는 표현 |

#### 인스타그램/틱톡 해시태그

| 분류 | 분석 기준 |
|-----|---------|
| 볼륨 | 게시물 수 (대형: 100만+, 중형: 10만~100만, 소형: 1만~10만) |
| 전략 | 대형 1개 + 중형 3개 + 소형 3~5개 조합 |
| 틈새 | 경쟁 낮고 UniPort 타겟이 쓰는 해시태그 |

### Step 3: 키워드 매트릭스 생성

```
| 키워드 | 예상 검색량 | 경쟁 강도 | 채널 적합성 | 우선순위 |
|--------|-----------|---------|-----------|---------|
| 모의투자 앱 | 높음 | 중간 | 네이버/구글/유튜브 | ⭐⭐⭐ |
| 20대 주식 시작 | 중간 | 낮음 | 네이버/유튜브 | ⭐⭐⭐ |
| Z세대 재테크 | 낮음 | 낮음 | 인스타/틱톡 | ⭐⭐ |
```

우선순위: ⭐⭐⭐ 즉시 활용 / ⭐⭐ 중기 전략 / ⭐ 장기 모니터링

### Step 4: 롱테일 키워드 발굴

단기 경쟁 피하고 전환율 높은 롱테일 키워드:

**패턴**:
- "[페르소나] + [문제] + [해결]": "대학생 주식 시작 어떻게"
- "[비교]": "모의투자 앱 어디가 좋아"
- "[방법]": "주식 처음 공부 어디서 시작"
- "[우려]": "주식 잃을까봐 무서운데 시작"

### Step 5: 트렌드 모니터링

**월간 모니터링 대상**:
- 네이버 DataLab 급상승 검색어
- 구글 트렌드 "주식", "재테크" 관련
- 에브리타임 실시간 인기글 키워드
- 유튜브 주식/재테크 카테고리 인기 검색어

**계절/이벤트 키워드** (선제 대응):
- 3월: "새내기 재테크" (신입생 시즌)
- 5월: "상반기 보너스 투자" (사회초년생)
- 9월: "2학기 용돈 재테크" (대학생)
