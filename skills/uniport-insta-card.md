# UniPort 인스타그램 카드뉴스

Z세대 투자 입문자를 위한 인스타그램 카드뉴스(캐러셀) 콘텐츠를 제작한다.

## 시작 전 컨텍스트 확인

반드시 먼저 읽는다:
- `marketing/brand-guide.md` — 브랜드 보이스, 금지 표현
- `marketing/target-persona.md` — 타겟 페르소나
- `marketing/product-faq.md` — 제품 핵심 기능

## 입력 수집

제공되지 않은 경우에만 물어본다:

1. **주제**: 어떤 투자/금융 주제를 다룰지 (예: "주식 용어 기초", "분산투자란?")
2. **타겟 페르소나**: A(대학생) / B(코인 Z세대) / C(사회초년생) 중 선택
3. **카드 수**: 5장 / 7장 / 10장 (기본값: 7장)
4. **톤**: 교육형 / 공감형 / 바이럴형

## 카드뉴스 제작 워크플로우

### Step 1: 주제 분석
- 타겟 페르소나가 실제로 궁금해할 각도로 접근
- 투자 전문용어 → Z세대 언어로 변환
- 핵심 메시지 1줄 정의

### Step 2: 슬라이드 구조 설계

**7장 기본 구조:**
| 슬라이드 | 역할 | 글자 수 |
|---------|------|--------|
| 1 | 훅 (공감 질문 또는 충격 통계) | 20자 이내 |
| 2 | 문제 제시 (페르소나 공감) | 30자 이내 |
| 3 | 핵심 개념 1 | 40자 이내 + 예시 |
| 4 | 핵심 개념 2 | 40자 이내 + 예시 |
| 5 | 핵심 개념 3 | 40자 이내 + 예시 |
| 6 | UniPort 연결 (자연스럽게) | 50자 이내 |
| 7 | CTA + 저장 유도 | 30자 이내 |

### Step 3: 훅 공식 (슬라이드 1)

**공감형**: "주식 시작하고 싶은데 뭐부터 해야 할지 모르겠어?"
**통계형**: "20대 70%가 재테크 필요하다고 느끼지만 시작은 못 함"
**질문형**: "PER이 뭔지 아직도 모름? 이 카드 저장해"
**반전형**: "주식 공부 3년 해도 돈 못 버는 이유"

### Step 4: CTA 공식 (슬라이드 마지막)

- "저장해두고 나중에 볼 때 쓰기 🔖"
- "투자 시작은 유니포트로 👉 링크 바이오"
- "친구한테 공유해줘 (도움될 사람 있을 거야)"

### Step 5: 캡션 작성

```
[훅 1문장]

카드 넘겨봐 👇

#주식입문 #재테크 #20대투자 #유니포트 #모의투자 #Z세대재테크 #투자시작 #주식공부
```

## 검증 체크리스트

- [ ] 슬라이드당 메시지 1개만 (여러 개 X)
- [ ] 투자 보장 표현 없음
- [ ] Z세대 언어 (전문용어 번역됨)
- [ ] 마지막 슬라이드에 저장/공유 유도 CTA
- [ ] 브랜드 보이스 준수

## 변형 옵션

요청 시 제공:
- **A/B 테스트 버전**: 훅만 다른 2가지 버전
- **틱톡 전환**: 같은 콘텐츠를 틱톡 스크립트로 변환
- **블로그 전환**: 카드뉴스 → 블로그 롱폼으로 확장
