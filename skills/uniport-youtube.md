# UniPort 유튜브 콘텐츠 생성

유튜브 쇼츠와 롱폼 스크립트, SEO 메타데이터, 썸네일 문구를 완성형으로 제작한다.

## 시작 전 컨텍스트 확인

반드시 먼저 읽는다:
- `marketing/brand-guide.md` — 브랜드 보이스
- `marketing/target-persona.md` — 타겟 페르소나
- `marketing/product-faq.md` — 제품 핵심 기능 및 차별점

## 입력 수집

제공되지 않은 경우에만 물어본다:

1. **형식**: 쇼츠(60초) / 롱폼(5~10분)
2. **주제**: 어떤 투자/UniPort 관련 주제
3. **타겟**: 페르소나 A/B/C
4. **목표**: 브랜드 인지도 / 앱 다운로드 유도 / 교육 콘텐츠

## 유튜브 쇼츠 워크플로우 (60초)

### Step 1: 훅 설계 (0~3초)

**목표**: 시청자가 스크롤을 멈추게 한다.

훅 공식:
- **질문형**: "주식 처음 시작하면 무조건 이거 알아야 해"
- **숫자형**: "20대 투자자 10명 중 8명이 이 실수 함"
- **반전형**: "주식 공부 많이 할수록 오히려 손해인 이유"
- **공감형**: "나 혼자만 이해 못 하는 거 아니었어"

### Step 2: 스크립트 구조 (60초)

```
[0~3초]  훅 — 시청자 멈추기
[3~15초] 문제/공감 — "맞지? 나도 이랬어"
[15~45초] 핵심 정보 — 2~3가지 포인트, 쉽게 설명
[45~55초] UniPort 연결 — 자연스럽게 "이걸 쉽게 해주는 앱이 있는데"
[55~60초] CTA — "링크 바이오에서 무료로 해봐"
```

### Step 3: 자막 스타일 지침

- 한 화면에 1~2문장만
- 강조 단어는 **굵게** 또는 다른 색
- 빠른 텍스트 전환 (Z세대 선호)
- 이모지 1~2개로 포인트

## 유튜브 롱폼 워크플로우 (5~10분)

### Step 1: 키워드 리서치

`/uniport-keyword-analysis` 스킬로 먼저 타겟 키워드 확인 (선택)

### Step 2: 영상 구조 (스크립트 800~1500단어)

```
[인트로 0~30초]     훅 + 영상에서 배울 것 예고
[파트 1]           문제/공감 (시청자가 겪는 상황)
[파트 2]           핵심 개념 설명 (쉬운 언어로)
[파트 3]           실제 예시 또는 시뮬레이션
[파트 4]           UniPort로 해결하는 방법 (자연스럽게)
[아웃트로 30초]     구독 유도 + CTA + 관련 영상 추천
```

### Step 3: 챕터 타임스탬프 (자동 생성)

```
00:00 인트로
00:30 [파트1 제목]
02:00 [파트2 제목]
04:30 [파트3 제목]
07:00 UniPort로 바로 시작하기
09:00 마무리
```

## SEO 메타데이터 생성

### 제목 공식 (60자 이내)

- **숫자 + 키워드 + 대상**: "주식 처음 하는 사람이 꼭 알아야 할 3가지 (2026년)"
- **질문형**: "주식 초보, 어디서 시작해야 해? 유니포트로 해봤다"
- **결과형**: "모의투자로 3개월, 실제로 배운 것들"

3가지 제목 옵션 항상 제공 (A/B 테스트용)

### 설명란 (SEO 최적화, 200자 + 영문 키워드)

```
[첫 두 줄 — 검색 미리보기에 보임]
주식 처음 시작하는 20대를 위한 완벽 가이드입니다.
유니포트 앱으로 리스크 없이 모의투자를 경험해보세요.

[본문]
이 영상에서는 ...을 다룹니다.

📲 유니포트 다운로드: [앱스토어 링크]

⏱ 챕터
00:00 ...

🔑 키워드: 주식입문, 모의투자, Z세대투자, 주식공부, 재테크

#주식입문 #모의투자 #유니포트 #재테크 #20대투자
```

### 태그 목록 (15~20개)

주제에 맞는 한국어/영어 혼합 태그 생성

### 썸네일 문구

3가지 옵션:
- 메인 텍스트 (20자 이내, 굵고 큰 글씨)
- 서브 텍스트 (선택, 10자 이내)
- 배경/분위기 제안

## 출력 형식

```markdown

### 📋 SEO 제목 옵션

3. [제목 C]

### 🎬 스크립트
[타임코드] [내용]
...

[완성된 설명란]

### 🏷️ 태그

### 🖼️ 썸네일 문구

```

## 콘텐츠 캘린더 연동

월간 유튜브 발행 계획:
- 쇼츠: 주 1회
- 롱폼: 격주 1회
- 콘텐츠 캘린더가 필요하면 `/content-strategy` 스킬 참고

## 검증 체크리스트

- [ ] 훅이 3초 안에 흥미를 끄는가
- [ ] 투자 수익 보장 표현 없음
- [ ] Z세대 언어 사용 (전문용어 번역)
- [ ] UniPort 언급이 자연스럽게 녹아 있음
- [ ] CTA가 명확함
- [ ] SEO 제목에 타겟 키워드 포함
